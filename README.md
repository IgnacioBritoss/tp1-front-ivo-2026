# Quiniela Britos
 Ignacio Britos 5A

## Descripción
Esta mini app permite:
- generar jugadas de quiniela,
- guardar boletas en un historial,
- ver estadísticas en tiempo real,
- cambiar entre modo oscuro y claro.

## Herramientas usadas
- Astro
- HTML
- CSS
- JavaScript

## Cómo ejecutarlo
1. Abrir una terminal en la carpeta del proyecto.
2. Instalar dependencias:
   ```bash
   npm install
   ```
3. Levantar el entorno local:
   ```bash
   npm run dev
   ```
4. Abrir la URL que muestra Astro en el navegador.
